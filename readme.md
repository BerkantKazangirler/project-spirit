# 🛰️ Project Spirit - Disaster Management Ecosystem

TUA Astro Hackathon 2026 kapsamında, **Spirit Dynamics** çatısı altında geliştirilen; afet anında internet bağımsız kriz yönetimi sağlayan dijital ekosistemin ana kontrol merkezidir.

## 🚀 Öne Çıkan Özellikler
- **3D Dashboard:** React 19 ve Mapbox GL JS ile afet bölgesinin (Antakya/Sümerler) anlık görselleştirilmesi.
- **Safe-Router:** AI'dan gelen verilerle enkaz ve riskli bölgeleri bypass eden dinamik rota hesaplama.
- **BKZS Entegrasyonu:** İnternet bağımsız Mesh Network üzerinden saha ekipleriyle veri senkronizasyonu.
- **Real-time Tracking:** Alpha ve Bravo ekiplerinin koordinat bazlı anlık takibi.

## 🛠️ Teknik Stack
- **Frontend:** Next.js 16.2, React 19, Vite 8
- **Styling:** Tailwind CSS
- **Maps:** Mapbox GL JS
- **State Management:** TypeScript & Context API
